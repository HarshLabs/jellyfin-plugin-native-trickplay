Native Trickplay

Generates HLS I-frame playlists so AVPlayer (Apple TV) gets native scrubbing thumbnails. Asset is built once per item and cached.

Generation

If you change this, existing cached assets at the old path become orphaned — delete them manually if you don't want to re-encode from scratch.

Cache pruning

Runs daily at 3 AM by default; can be triggered manually from Dashboard → Scheduled Tasks → Native Trickplay.
