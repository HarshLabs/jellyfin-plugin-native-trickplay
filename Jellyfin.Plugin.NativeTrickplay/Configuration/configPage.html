Native Trickplay

Generates HLS I-frame playlists so AVPlayer-based clients (Apple TV) get native scrubbing thumbnails on the transport bar. Each item is encoded once and cached on disk.

Generation

How thumbnails are encoded
Enable plugin Turn off to disable trickplay generation and master-playlist injection. Existing cached assets are kept.
Use hardware decoding Follows Dashboard → Playback → Transcoding's hwaccel setting. Big speedup for HEVC/HDR sources; encoding stays on CPU.
Thumbnail height Pixels — width is auto-scaled to the source's aspect ratio.
Thumbnail interval Seconds between scrubbing thumbnails. 1 = smoothest scrubbing; raise to 5 or 10 if disk space is tight.
Encoder quality (CRF) Lower number → higher quality, larger files. 18–32 typical.
Cache directory Absolute path. Leave blank to use Jellyfin's default cache. If you change this, old caches at the previous path are orphaned — delete manually.

Pruning

Runs daily at 3 AM by default · Dashboard → Scheduled Tasks → Native Trickplay
Remove orphaned caches Drop cache entries whose item is no longer in the library.
Age-based eviction Delete entries unaccessed for more than N days. 0 disables.
Total size cap Keep total cache below N GB by LRU eviction. 0 disables.
Changes take effect immediately for new playbacks.
