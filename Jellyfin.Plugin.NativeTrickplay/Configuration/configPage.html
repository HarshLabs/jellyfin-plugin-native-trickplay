Native Trickplay

Generates HLS I-frame playlists so AVPlayer (Apple TV) gets native scrubbing thumbnails. Asset is built once per item and cached.

Generation

1 = one thumbnail per second of source (smoothest scrubbing, ~30 MB cache for a 25-min episode at 320p). Increase to 5 or 10 if disk space matters more than scrubbing granularity.
If you change this, existing cached assets at the old path become orphaned — delete them manually if you don't want to re-encode from scratch.

Cache pruning

Runs daily at 3 AM by default; can be triggered manually from Dashboard → Scheduled Tasks → Native Trickplay.

Cache Management

Inspect cache state and selectively trigger trickplay generation for specific items or whole libraries. Generation runs in the background; once started, scrubbing thumbnails appear as soon as the encode finishes (typically 1–6 minutes per item depending on resolution and source codec).
